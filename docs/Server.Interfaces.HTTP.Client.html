Class: Client

Server.Interfaces.HTTP. Client

new Client()

This is the HTTP Client class. It does not need to be instantiated, so you can safely ignore the constructor. You can access methods on it in this way - "interface.client.method". This class contains methods to make outgoing HTTP requests and receive responses.
